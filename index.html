01

Sistema Informático

▼

Conceptos Fundamentales

  • Informática: Es la ciencia que estudia el tratamiento automático de la información mediante computadoras
  • Sistema y Subsistema: Un sistema informático es un conjunto de elementos que trabajan juntos
  • Los 5 elementos son: físico (hardware), lógico (software), humano (usuarios), dato (entrada) e información (salida)

Evolución Tecnológica

  • Generaciones de computadoras: Desde las válvulas de vacío hasta la inteligencia artificial
  • 1° Generación: Válvulas (1940-1956)
  • 2° Generación: Transistores (1956-1963)
  • 3° Generación: Circuitos integrados (1964-1971)
  • 4° Generación: Microprocesadores (1971-presente)
  • 5° Generación: Inteligencia Artificial (presente-futuro)

Hardware (Recursos Físicos)

  • Periféricos de entrada: teclado, mouse, micrófono, scanner
  • Periféricos de salida: monitor, impresora, parlantes
  • Periféricos de E/S: pantallas táctiles, impresoras multifunción
  • Almacenamiento: HDD, SSD, USB, tarjetas de memoria
  • Unidades: Bit, Byte, KB, MB, GB, TB

Software (Recursos Lógicos)

  • Software de Sistema: Windows, Linux, macOS
  • Software de Programación: Python, Visual Studio, compiladores
  • Lenguajes: Alto nivel (Python, Java) vs Bajo nivel (Ensamblador)
  • Software de Aplicación: Word, Excel, Chrome, Photoshop
02

Sistema Operativo

▼

Fundamentos del SO

  • ¿Qué es? Software que gestiona el hardware y permite que funcionen otros programas
  • Sesión de trabajo: Inicio de sesión → Trabajo → Cierre de sesión
  • El Escritorio: Tu espacio de trabajo visual

Elementos de la Interfaz

  • Botones: Inicio, cerrar, minimizar, maximizar
  • Iconos: Representación gráfica de archivos y programas
  • Ventanas: Espacios donde se ejecutan las aplicaciones

Explorador de Windows

  • Unidades: C:, D:, USB, discos externos
  • Carpetas: Contenedores para organizar archivos
  • Archivos: Documentos, imágenes, videos, etc.
  • Operaciones: Crear, copiar, mover, eliminar, renombrar
  • Búsqueda: Encontrar archivos rápidamente
03

Procesador de Texto

▼

Operaciones Básicas

  • Crear documentos: Diseño y planificación del contenido
  • Edición: Suprimir, trasladar, copiar e insertar texto
  • Guardar: .doc, .docx, .pdf
  • Imprimir: Configuración de página y vista previa

Formato de Texto

  • Fuentes: Arial, Times New Roman, tamaño, color
  • Estilos: Negrita, cursiva, subrayado
  • Espaciado: Interlineado, espaciado entre párrafos
  • Alineación: Izquierda, centro, derecha, justificado

Formato de Párrafo

  • Sangría: Primera línea, francesa, izquierda, derecha
  • Tabulaciones: Organizar texto en columnas
  • Viñetas y numeración: Listas ordenadas y desordenadas

Herramientas Avanzadas

  • Tablas: Organizar información en filas y columnas
  • Crear, editar, ordenar y dar formato a tablas
  • Imágenes: Insertar, ajustar tamaño, posición
  • Relación texto-imagen: Ajuste de texto alrededor
  • Gráficos: Torta, barras para visualizar datos
  • Hipervínculos: Enlaces a páginas web o documentos
  • Combinación de correspondencia: Documento principal + fuente de datos
  • Columnas: Texto estilo periodístico

Presentaciones Multimedia

  • PowerPoint: Herramienta para presentaciones
  • Plantillas y diseños: Autocontenido y autodiseño
  • Vistas: Normal, clasificador, presentación
  • Animaciones: Transiciones entre diapositivas
  • Multimedia: Imágenes, tablas, audio y video
04

Desarrollo Web (HTML & CSS)

▼

Conceptos Web

  • Internet: Red mundial de computadoras conectadas
  • Web: Servicio de páginas conectadas por hipervínculos
  • Navegadores: Chrome, Firefox, Edge, Safari

HTML - Estructura

  • Estructura básica: <html>, <head>, <body>
  • Encabezados: <h1> hasta <h6>
  • Párrafos: <p>
  • Listas: <ul> (desordenadas), <ol> (ordenadas), <li>
  • Hipervínculos: <a href="url">
  • Imágenes: <img src="ruta" alt="descripción">
  • Videos: <video>

Tablas en HTML

  • <table>: Define la tabla
  • <tr>: Fila de la tabla
  • <td>: Celda de datos
  • <th>: Celda de encabezado

Formularios

  • <form>: Contenedor del formulario
  • <input>: Campos de entrada (text, email, password)
  • <label>: Etiquetas para los campos
  • <button>: Botón de envío

CSS - Estilo

  • Separación: HTML (estructura) vs CSS (presentación)
  • Formas de aplicar: Inline, <style>, archivo externo (.css)
  • Selectores: tipo, .clase, #id

Box Model

  • Content: El contenido del elemento
  • Padding: Espacio interno
  • Border: Borde del elemento
  • Margin: Espacio externo

Propiedades CSS

  • Tipografía: font-family, font-size, color
  • Flexbox: display: flex para organizar elementos
  • Responsive: viewport y media queries para móviles
05

Planilla de Cálculo

▼

Conceptos Fundamentales

  • Modelización: Representar situaciones del mundo real con datos
  • Análisis de datos: Procesar información para tomar decisiones
  • Validación: Verificar que el modelo funcione correctamente

Estructura de Excel

  • Libro: Archivo de Excel (.xlsx)
  • Hojas: Pestañas dentro del libro
  • Filas: Numeradas (1, 2, 3...)
  • Columnas: Letradas (A, B, C...)
  • Celdas: Intersección de fila y columna (A1, B5, C10)
  • Rangos: Grupo de celdas (A1:C5)

Tipos de Datos

  • Texto: Palabras y letras
  • Números: Enteros y decimales
  • Fechas: dd/mm/aaaa
  • Fórmulas: Comienzan con =

Operaciones

  • Básicas: Copiar, cortar, pegar, rellenar
  • Formato: Negrita, color, bordes, combinar celdas
  • Ordenar: Ascendente, descendente
  • Filtrar: Mostrar solo ciertos datos

Fórmulas y Funciones

  • Operadores: + (suma), - (resta), * (multiplicación), / (división)
  • SUMA: =SUMA(A1:A10)
  • PROMEDIO: =PROMEDIO(A1:A10)
  • MAX/MIN: Valor máximo y mínimo
  • CONTAR: Cuenta celdas con números
  • SI: =SI(condición, verdadero, falso)
  • Referencias: Relativas (A1) y Absolutas ($A$1)

Gráficos

  • Tipos: Columnas, líneas, torta, barras
  • Elementos: Título, ejes, leyenda
  • Seleccionar datos → Insertar → Gráfico
06

Bases de Datos

▼

Conceptos Básicos

  • Base de Datos: Colección organizada de información
  • Tabla: Estructura que contiene datos relacionados
  • Registro: Fila de la tabla (una entrada completa)
  • Campo: Columna de la tabla (un tipo de dato)

Tipos de Bases de Datos

  • Relacionales: Datos organizados en tablas relacionadas
  • No Relacionales: Documentos, gráficos, clave-valor

SGBD - Sistemas de Gestión

  • ¿Qué es? Software para crear y administrar bases de datos
  • Motores: SQLite, Microsoft SQL Server, MySQL, PostgreSQL
  • Funciones: Crear, consultar, modificar, eliminar datos

SQL - Lenguaje de Consulta

  • SQL: Structured Query Language
  • Historia: Creado en los años 70 por IBM
  • Es el lenguaje estándar para bases de datos relacionales

Sentencia SELECT

  • Estructura básica: SELECT campos FROM tabla WHERE condición
  • Ejemplo 1: SELECT * FROM Alumnos
  • Ejemplo 2: SELECT Nombre, Edad FROM Alumnos WHERE Edad > 15
  • FROM: Especifica la tabla
  • WHERE: Filtra los resultados

Práctica con SGBD

  • Instalación: Configurar SQL Server o MySQL
  • Entorno de trabajo: Interfaz gráfica o línea de comandos
  • Servicios: Iniciar y detener el motor de BD
  • Crear tablas: Definir estructura y tipos de datos
  • Consultar: Usar SELECT para obtener información
